## DELIVERABLE 1: PRESENTER NOTES

### Requirements (from step5_presentation_standards.txt + pipeline_config.json)

| Parameter | Requirement |
|-----------|-------------|
| Format | Verbatim monologue (complete sentences) |
| Word Count | 130-150 words per minute of delivery |
| Maximum Duration | 180 seconds per slide |
| Maximum Words | ~450 words per slide |
| Minimum Duration | 45 seconds per slide |
| Minimum Words | ~130 words per slide |

### Required Elements Per Slide
- [ ] Opening statement for the slide
- [ ] Explanation of EACH concept on the slide
- [ ] Applied relevance or significance statements
- [ ] Connection to previous or upcoming content
- [ ] Closing transition to next slide
- [ ] [PAUSE] markers for deliberate pauses
- [ ] [EMPHASIS: term] markers for key terms
- [ ] NCLEX pattern callouts where applicable

### Tone Requirements (from presenter_notes_tone_guidelines.txt)

| Element | Frequency |
|---------|-----------|
| Dramatic urgency language | 2-3 times per section |
| Fear-based motivation (exam) | 3-4 times per section |
| Fear-based motivation (clinical) | 1-2 times per section (for dangerous content) |
| Pattern interrupts | Every 3-4 slides |
| Integration callbacks | 1-2 times per section |

### Required Language Patterns
- [ ] "life-threatening", "absolutely essential", "fatal if not recognized"
- [ ] "Here's what you need locked in for exam day"
- [ ] "The NCLEX tests this constantly"
- [ ] "If you miss this, you've lost an easy 2 points"
- [ ] "Get this wrong, and your patient dies" (only for dangerous content)
- [ ] Rhetorical questions ("Why does this matter?")
- [ ] Dramatic reveals ("Here's the key: [revelation]")

### Quality Check
- [ ] Not every sentence is dramatic (causes habituation)
- [ ] Fearmongering used strategically, not constantly
- [ ] Clear explanations mixed with urgency
- [ ] Logical flow maintained

## DELIVERABLE 2: FIXED SLIDES PER SECTION

### Section Intro Slide
- [ ] Position: First slide of section
- [ ] Header: Section title
- [ ] Body: Poetic/provocative quote
- [ ] NCLEX Tip: None
- [ ] Presenter Notes: Welcome + preview + connection to prior content

### Vignette Slide
- [ ] Position: Near section end (before answer)
- [ ] Format: NCLEX-style clinical vignette
- [ ] Stem: 2-4 sentences
- [ ] Options: A, B, C, D (4 plausible answers)
- [ ] Presenter Notes: Read aloud + allow think time (30-60 sec)
- [ ] DO NOT reveal answer on this slide

### Answer Slide
- [ ] Position: Immediately after vignette
- [ ] Header: "Answer: [Letter]"
- [ ] Body: Correct answer + rationale + distractor analysis
- [ ] Presenter Notes: Explain correct + walk through distractors

## SHAPE ROTATION GUIDELINES (for visual aids)

**CRITICAL**: PowerPoint rotation happens around the shape's CENTER, not top-left corner.

For lines/arrows:
1. Calculate midpoint between start and end
2. Position shape so CENTER is at midpoint
3. Then rotate

**Use helper functions**: `draw_line()`, `draw_arrow()` from step12_powerpoint_population.py
