--------------------------------------------------------------------------------

- Title positioning: 0.3" from top of content area (if overall title needed)


================================================================================
IMPLEMENTATION NOTES
================================================================================

CONTENT AREA BOUNDARIES
--------------------------------------------------------------------------------
All graphic organizers must fit within white content rectangle:
- Left: 0.54"
- Top: 0.72"
- Width: 12.26"
- Height: 6.52"
- Margins: 0.3" from edges (usable area: 11.66" x 5.92")

TITLE HANDLING
--------------------------------------------------------------------------------
- Original slide title: Move to template header TextBox 2
- Shorten to ≤28 characters if needed
- Title font: 24pt Bold White (in dark header area)

TEXT OVERFLOW HANDLING
--------------------------------------------------------------------------------
- If text exceeds shape bounds: Reduce font to 16pt minimum
- If still overflowing: Expand shape height (not width)
- Maintain minimum 18pt for final output

TIER BADGE
--------------------------------------------------------------------------------
- Keep original "TIER 1" badge from template (or update based on section)
- Position: Top-right (11.49", 0.13")

SHADOW RENDERING
--------------------------------------------------------------------------------
- Use PowerPoint's built-in shadow presets where possible
- Outer shadows: For depth and separation
- Inner shadows: For recessed effects (spectrum bar)
- Glow: For emphasis (timeline markers, key differentiators)

3D BEVEL TYPES
--------------------------------------------------------------------------------
- Soft Round: Gentle, professional (most common)
- Angle: Sharp, modern (hierarchy top level, decision diamonds)
- Emboss: Subtle raised effect (labels, small elements)

COLOR ACCESSIBILITY
--------------------------------------------------------------------------------
- All color combinations must maintain 4.5:1 contrast ratio minimum
- Black text on light backgrounds (always)
- White text only on dark backgrounds (headers, some hierarchy boxes)
